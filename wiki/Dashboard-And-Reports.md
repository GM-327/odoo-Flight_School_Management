# Dashboard & Reports

Analytics and reporting features in the Flight School Management System.

---

## Dashboard Overview

**Navigation:** Flight School → Dashboard

The dashboard provides at-a-glance metrics:

### Key Performance Indicators

| Metric | Description |
|--------|-------------|
| Active Students | Currently enrolled students |
| Available Aircraft | Aircraft ready for flight |
| Instructors on Duty | Available instructors |
| Flights Today | Scheduled flights for today |

### Status Widgets

- **Fleet Status** - Aircraft availability chart
- **Upcoming Expirations** - Licenses and medicals due soon
- **Maintenance Due** - Aircraft needing service
- **Recent Activity** - Latest system actions

## Generating Reports

### Step-by-Step

1. Navigate to **Flight School → Reports**
2. Select report category
3. Choose specific report
4. Set parameters:
   - Date range
   - Filters (aircraft, student, etc.)
   - Grouping options
5. Click **Generate**

### Export Options

| Format | Use Case |
|--------|----------|
| PDF | Printing, formal records |
| Excel | Data analysis, manipulation |
| CSV | Import to other systems |

---

## Custom Reports

### Using Filters

Apply filters to any list view:
1. Click **Filters** dropdown
2. Add filter conditions
3. Save as custom filter

### Grouping Data

Group records by:
- Date (day, week, month)
- Status
- Category
- Assigned person

### Pivot Tables

For advanced analysis:
1. Switch to **Pivot** view
2. Drag fields to rows/columns
3. Select measures
4. Export as needed

---

## Scheduled Reports

### Setting Up Automatic Reports

1. Configure report parameters
2. Click **Schedule** (if available)
3. Set frequency:
   - Daily
   - Weekly
   - Monthly
4. Enter recipient emails
5. Save schedule

### Common Scheduled Reports

| Report | Frequency | Recipients |
|--------|-----------|------------|
| License Expirations | Weekly | Training Manager |
| Maintenance Due | Daily | Maintenance Chief |
| Fleet Status | Daily | Operations Manager |

---

## Dashboard Customization

### Adding Widgets

1. Edit dashboard (if permitted)
2. Click **Add Widget**
3. Select widget type
4. Configure data source
5. Save

### Available Widget Types

| Type | Description |
|------|-------------|
| Number | Single metric display |
| Chart | Bar, line, pie charts |
| List | Recent records |
| Calendar | Upcoming events |

---

## Data Export

### Bulk Export

1. Go to any list view
2. Select records (or select all)
3. Click **Action → Export**
4. Choose fields to export
5. Select format
6. Download file

## [19.0.1.0.0] - 2024-12-01

### Added
- **fs_core**: Core settings module
  - Security groups (User, Instructor, Manager, Admin)
  - Central configuration
  - Module category definition
  
- **fs_fleet**: Fleet management module
  - Aircraft registry
  - Maintenance tracking
  - Flight hours logging
  
- **fs_people**: Personnel management module
  - Student management
  - Instructor profiles
  - License tracking
  - Medical certificate tracking

### Technical
- Built on Odoo 19.0
- LGPL-3.0 license
- EASA regulation compliance framework

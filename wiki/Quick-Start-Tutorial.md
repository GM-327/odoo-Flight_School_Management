# Quick Start Tutorial

Get up and running with the Flight School Management System in 15 minutes!

## Prerequisites

- System installed and running (see [Installation Guide](Installation-Guide))
- Admin access to the system
- A web browser

---

## Step 1: First Login (2 min)

1. Open your browser and go to `http://localhost:8069`
2. Login with your admin credentials
3. You'll see the Odoo main menu

---

## Step 2: Install Flight School Modules (3 min)

1. Go to **Apps** menu
2. Remove the "Apps" filter in the search bar
3. Search for "flight"
4. Install these modules in order:
   - `Flight School Settings` (fs_core)
   - `Flight School Fleet` (fs_fleet)
   - `Flight School People` (fs_people)

---
